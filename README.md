# Java Reactive Challenges & Concepts

Welcome to this repository! 🎯

This space is intended to host different **proofs of concept** and **coding challenges** focused on Java, particularly exploring modern patterns and paradigms such as **Reactive Programming**, **Functional Programming**, and **clean design practices**.

---

## 📁 Structure

This repository will contain a collection of standalone folders or modules, each representing a different challenge, concept, or learning goal. Currently available:

### 1. [`movies-manager/`](movies-manager/README.md)
A reactive programming challenge based on a movies dataset, where you can query and transform data about:
- 🎬 Movies
- 🎭 Actors
- 🎥 Directors
- 📊 Ratings
- 🧩 Categories

Includes medium and advanced-level exercises using `Flux`, `Mono`, `Mono.zip`, grouping, and filtering.

---

## 🧠 Goal

The purpose of this repository is to:

- Help students and developers practice key concepts of Java and reactive programming.
- Serve as a playground for trying out new libraries, coding techniques, or design approaches.
- Enable reusable, hands-on exercises for learning and teaching.
